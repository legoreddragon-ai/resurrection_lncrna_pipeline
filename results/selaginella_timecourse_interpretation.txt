S. lepidophylla lncRNA Time-Course Kinetics
============================================
Dataset: PRJNA420971 — 0hr, 1hr, 6hr, 24hr, 120hr desiccation + rehydration

Two induction patterns identified:

Early responder (1hr activation):
  MSTRG.22821: 0 → 3.6 → 1.2 → 18.7 → 2.9 → 9.1 TPM
  Active from 1hr, peaks at 24hr, sustained in rehydration
  Consistent with early alarm/signalling role

Late responders (24hr+ activation):
  MSTRG.15112: 0 → 0 → 1.0 → 16.2 → 10.3 → 10.3 TPM
  MSTRG.15115: 0 → 0 → 0.9 → 14.8 → 11.0 → 11.6 TPM
  MSTRG.15111: 0 → 0 → 0.8 → 14.1 → 10.3 → 9.9 TPM
  MSTRG.6238:  0 → 0 → 0.0 → 12.2 → 10.6 → 8.3 TPM
  Silent until severe desiccation (24hr), sustained through 120hr
  Consistent with roles in prolonged survival and recovery

Key finding: S. lepidophylla lncRNA programme has temporal structure —
early and late responding subsets with distinct kinetic profiles.
This temporal organisation is not resolvable in C. plantagineum due
to the single desiccation time point available in GSE157098.
